# Konimbo User File Code

Files in this directory contain code to paste into Konimbo admin user_files.

## Setup Instructions

### 1. Create user_file: `aluf-ui-shell`
- **Type:** חנות (store)
- **Target:** כל החנות (layout)
- **Active:** Yes
- Copy contents from:
  - `aluf-ui-shell.css` → css_before field
  - `aluf-ui-shell.head.html` → head_html_before field
  - `aluf-ui-shell.foot.html` → foot_html_before field
  - `aluf-ui-shell.js` → javascript_before field

### 2. Create user_file: `aluf-page-home`
- **Type:** חנות (store)
- **Target:** דף הבית (homepage)
- **Active:** Yes
- Copy contents from:
  - `aluf-page-home.foot.html` → foot_html_before field
