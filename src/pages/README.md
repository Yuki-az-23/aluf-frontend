# Pages

Each file is a full page component that composes layout and commerce components.

- `HomePage.tsx` — Homepage with hero, services, products, tiers, blog, newsletter
- `CategoryPage.tsx` — Category subcategory grid (stub)
- `ItemsGridPage.tsx` — Product listing with filters (stub)
- `ItemPage.tsx` — Product detail page (stub)
- `CartPage.tsx` — Shopping cart (stub)

Pages are rendered by the router in `App.tsx` based on `<meta name="aluf-page">`.
